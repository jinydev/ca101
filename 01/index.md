# 컴퓨터 시스템 개요





## 학습 목표

* 컴퓨터의 기본적인 구조에 대해 학습한다.
* 정보를 어떻게 표현하고 저장하는지 알아본다.
* 컴퓨터 시스템이 어떻게 구성되는지 살펴본다.





## 목차

* 컴퓨터의 기본 구조
* 정보의 표현과 저장
* 시스템의 구성
